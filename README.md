# Harmony Station GitHub Pages Website

This is a static GitHub Pages project for Harmony Station Piano Lessons.

## Files

- `index.html` — main website page
- `styles.css` — visual styling and responsive layout
- `script.js` — mobile menu behavior

## How to preview locally

Open `index.html` directly in a browser.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `styles.css`, and `script.js` to the repository root.
3. Go to **Settings → Pages**.
4. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/ (root)`
5. Save.

## Custom domain

If you want to use an existing domain such as `harmonystation.info`, add a `CNAME` file containing only the domain name, then configure DNS with your domain provider according to GitHub Pages domain instructions.
